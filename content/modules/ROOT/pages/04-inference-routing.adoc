= Sovereign Inference
:navtitle: Sovereign Inference

*Time: ~10 minutes* · *Role: AI Engineer, Security Architect*

== Learning Objectives

* Send prompts through the sovereign inference stack
* Observe semantic routing decisions (sensitive, general, injection)
* Verify routing events are hash-chained in the ledger

== Exercise 1: General query

[source,bash,role="execute",subs=attributes+]
----
curl -sf -X POST {demo_api_url}/api/route \
  -H "Content-Type: application/json" \
  -d '{"prompt": "What is the general framework for AI governance?", "max_tokens": 100}' | python3 -c "
import sys, json
d = json.load(sys.stdin)
print(f'Route:    {d[\"route\"]}')
print(f'Model:    {d.get(\"model\", \"n/a\")}')
print(f'Response: {d.get(\"response\", \"\")[:200]}')
"
----

Expected: route `completed`, model `granite-3.2-sovereign`.

== Exercise 4: Check routing events in the ledger

[source,bash,role="execute",subs=attributes+]
----
curl -sf {demo_api_url}/api/ledger | python3 -c "
import sys, json
entries = json.load(sys.stdin)
router = [e for e in entries if e.get('entry_type', '').startswith('router.')]
print(f'Total routing events: {len(router)}')
print()
for e in router[-5:]:
    print(f'  {e[\"entry_type\"]:45s} {e[\"entry_hash\"][:16]}...')
"
----

Every routing decision — classify, route, block — is recorded in the chain.
