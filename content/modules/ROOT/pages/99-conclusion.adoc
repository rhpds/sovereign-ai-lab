== Run It Yourself

Fork the repository and run the local simulation stack:

[source,bash]
----
git clone https://github.com/jkershawrh/sovereign-ai-lab.git
cd sovereign-ai-lab
make up
----

For the guided OpenShift demo path, deploy the full stack to Oberon:

[source,bash]
----
NS=sovereign-ai-lab make deploy-oberon
----
